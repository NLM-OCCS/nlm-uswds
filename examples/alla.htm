How to use this template

This is a template for the page with NLM Header and Footer, designed for USWDS framework.

If you already have USWDS package in use

1. Add folder "nlm-fixes" to "scss" folder in your USWDS package.
You can find it in ⁨uswds⁩ › ⁨scss⁩

2. Add this code to the bottom of your uswds.scss file

// NLM//
@import "nlm-fixes/nlm-fixes";

3. Compile Sass

Overwise, you can use the whole package to start your project.
